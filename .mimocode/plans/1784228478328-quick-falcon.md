# Plan: Admin-Only Admin Creation

## Goal
Only existing admins can create new admin accounts. Customers cannot self-register as admin.

## Changes

### 1. Update `client/src/pages/Register.tsx`
- Remove the role selection dropdown
- All registrations default to "customer" role
- No option to choose "admin"

### 2. Add Admin Management Page
- Create `client/src/pages/AdminUsers.tsx`
- List all users
- Button to create new admin
- Form: name, email, password
- API call to POST /api/auth/create-admin

### 3. Add Route in `client/src/App.tsx`
- Add `/admin/users` route under admin protection

### 4. Update `client/src/pages/Dashboard.tsx`
- Add "Users" navigation link

### 5. Add Backend Route
- POST /api/auth/create-admin (admin only)
- Only admin role can access this endpoint

### 6. Update `server/src/controllers/authController.ts`
- Add createAdmin function
- Verify requesting user is admin
- Create new user with admin role

### 7. Update `server/src/routes/authRoutes.ts`
- Add create-admin route with admin middleware

## Verification
1. Try to register as admin → Should fail (only customer allowed)
2. Login as admin → See "Users" link in dashboard
3. Create new admin from dashboard
